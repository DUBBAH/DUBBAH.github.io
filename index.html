What is DUBBAH?

A project which focuses on developing publicly available machine learning algorithms that can assist providers in the management of the process of loaning medical aids.

The project is a partnership between Aarhus Unviersity, Kommunernes Landsforening, Aalborg Kommune, Aarhus Kommune, University College Nordjylland, and DigiRehab A/S. The project runs from December 2018 to October 2019 where development takes place at Aarhus Universty and is scheduled for the first 6 months. The following 6 months contains integration and pilot testing in Aalborg Kommune.
